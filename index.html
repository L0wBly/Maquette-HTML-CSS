Dévelopement Front

Le Html :

Le HTML sert à créer le contenue du site : elle va contenir uniquement la partie textuelle du site

Le Css :

Le CSS va vous permettre d'ajouter le couche graphique à notre site : couleur, forme fond, etcs..

Le JavaScript :

Javascript est un langage de programmation de script employé pour rendre une page web interactive. le comportement de la page et son apparence pourra varier en fonction de certains paramètres.

Le ReactJs :

React est une bibliothèque JavaScript libre dont le but est de faciliter la création d'application web monopage, via la création de composants d'interface dynamiques.

Dévelopement Back

PHP :

Le PHP est un langage de programmation libre, utilisé pour généré des pages web dynamiques, qui sont done modifiés en fonction de certaines informations (heure, nom de l'utilisateur, formulaire rempli par l'utilisateur etc).

Node :

Node,js est une plateforme logicielle libre et événementielle en JavaScript orientée vers les applications réseau qui doivent pouvoir monter en charge.

Python :

Python est un langage de programmation interprété, multi-paradigme et multiplateformes. Il favorise la programmation impérative structurée, fonctionnelle et orientée objet.
